# TCF7L2-MENA-Pharmacogenomics

Independent in-silico pharmacogenomic analysis exploring population-specific regulatory mechanisms of TCF7L2 variants associated with Type 2 Diabetes in Middle Eastern populations.

## Project Focus
- Population-specific allele frequency analysis
- Linkage disequilibrium (LD) analysis
- eQTL interpretation
- ENCODE/UCSC regulatory annotation
- Functional interpretation of rs12243326
- Pharmacogenomic integration using PharmGKB

## Key Findings
- Enrichment of TCF7L2 risk alleles in MENA populations
- Tissue-specific regulatory effects in vascular and adipose tissues
- Identification of a potentially independent regulatory signal at rs12243326
- Evidence supporting population-specific genomic architecture
